# Sofia Marketing Event - Landing Page Spec

## Project Context
Premium landing page for a live marketing transformation event in Sofia, Bulgaria. Target audience: established business owners, marketers, and personal brands seeking to scale.

## Hero Section Design

### Video-First Approach
- **Style**: Cinematic, atmospheric, dark with golden/amber light accents
- **Mood**: High-energy, transformational, premium corporate event
- **Visual**: Abstract golden light rays, particles, dynamic movement suggesting transformation and acceleration

### Color Palette (NO BLUE/PURPLE)
- **Primary Background**: Deep black (#0a0a0a)
- **Secondary**: Rich charcoal (#1a1a1a)
- **Accent Primary**: Warm gold (#D4AF37, #F4C430)
- **Accent Secondary**: Electric orange (#FF6B35)
- **Text Primary**: Pure white (#FFFFFF)
- **Text Secondary**: Soft cream (#F5F5DC)

### Typography
- **Headlines**: Extra bold, large scale (clamp responsive)
- **Main Title**: 4-6rem, uppercase tracking
- **Subtitle**: 1.5-2rem, lighter weight
- **Body**: Clean sans-serif, high readability

### Layout
- Full viewport hero (100vh)
- Centered content with max-width container
- Event badges floating/positioned strategically
- CTA prominent and urgency-driven

### Motion Design
- Subtle parallax on scroll
- Text reveal animations (staggered)
- Floating badges with gentle pulse
- Gradient shimmer on CTA button
- Video overlay with animated gradient

### Event Badges
- 220 места (seats)
- София (Sofia)
- Целодневно (Full-day)

### CTA
- Primary: "Вземи билет сега" with arrow
- Hover: Scale + glow effect
